# Global Power Plant EDA

## Overview  
This project explores the Global Power Plant Database to understand how energy is produced worldwide and compare renewable and non-renewable sources.

---

## Dataset  
- Source: Kaggle (World Resources Institute)  
- 30,000 rows, 24 columns  

---

## Data Cleaning  
- Removed columns with many missing values  
- Filled remaining missing values with "Unknown"  
- Checked for duplicates  

---

## Visualizations  
- Fuel type distribution  
- Renewable vs non-renewable  
- Capacity distribution  
- Top countries by number of plants  
- Top countries by total capacity  
- Capacity by fuel type  
- Country vs fuel heatmap  

---

## Key Insights  
- Energy production is shifting toward renewables, but non renewable fuels are still heavily used  
- Power plants are concentrated in a few major countries  
- Most plants have low capacity, while a few have very high capacity  

---
